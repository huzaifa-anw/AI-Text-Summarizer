# AI Text Summarizer

A simple AI text summarization web app built with HTML, CSS, JavaScript, Express.js, and the Hugging Face Inference API.

Paste any text into the input box, click **Summarize**, and the application generates a concise summary using the `facebook/bart-large-cnn` model.

## Features

- Summarizes long pieces of text
- Simple and responsive interface
- Uses the Hugging Face Inference API
- Input validation
- Loading indicator while the summary is generated
- Basic error handling

## What I Practiced

- Making POST requests with the Fetch API
- Building a simple Express API
- Using environment variables with dotenv
- Integrating an AI model through the Hugging Face Inference SDK
- Sending and receiving JSON between the frontend and backend
